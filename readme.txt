Video Gallery is an application to play video files.

It is built with Pyside and webkit.

This is very alpha so use with care.

Profiles must obey the following folder structure:

Series folder:
    Andromeda
        folder.png      - 200*100
        folder_big.png  - 
        s1
            1.avi
            2.avi
        

Movies folder:
    Avatar
        folder.png      - 200*100
        folder_big.png  - 
        movie.avi


Docs folder:
    BBC - Planet Earth
        folder.png      - 200*100
        folder_big.png  - 
        doc.avi


Sports folder:
    F1
        2009
            folder.png - 200*100
            04 - F1 - Bahrain GP
                race.avi
